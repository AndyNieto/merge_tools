Use the scripts in this folder to merge RET and Samsung files. 

Save the "Data_Collection" folder under "~/Documents", otherwise the scripts will require to be updated with the new location.

There are three python script inside this folder.

  > 1_GET_RET_reports.py = Use to download RET files. All files will be downloaded to the "input" directory
  > 2_MERGE_RET_reports.py = Use to merge RET files. The output will be saved under under a folder named under today's date

  > merge_USMsamsung.py = will merge any files inside the "./input/samsung/" directory. Make sure all the files are saved as ".xlsx"
